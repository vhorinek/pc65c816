pc65C816
========

Project features a computer based on W65C816 microprocessor. The computer is
divided into modules mounted in a custom 3D printed rack.

Computer features
-----------------

- W65C816 microprocessor
- MMU (with protection)
- rich IO (UART, SPI, I2C, IDE, Ethernet, ...)
